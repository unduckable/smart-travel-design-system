## Overview

Avatar là một yếu tố graphic để đại diện cho 1 user sử dụng sản phẩm, được thể hiện qua những element như ảnh đại diện, icon hoặc các chữ cái.

<Canvas of={Stories.Default} />

## Variants

### Size

Avatar có 3 size, sử dụng trong từng trường hợp cụ thể (vd: chatbox, homepage, profile page..)

<img src={image28} class="s-mb-4" style={{ maxWidth: 336 }} />

### Avatar dạng place holder

Sử dụng khi user chưa đăng nhập (guest)

<img src={image29} class="s-mb-4" style={{ maxWidth: 336 }} />

### Avatar text

Sử dụng khi user không upload ảnh, lấy theo chữ cái tên của user, màu sắc random theo bộ màu của Smart Travel

<img src={image30} class="s-mb-4" style={{ maxWidth: 336 }} />

### Avatar with indicator

Avatar chứa các indicator chỉ thị trạng thái online, hoặc có noti

<img src={image31} class="s-mb-4" style={{ maxWidth: 336 }} />

### Avatar group

Thể hiện một group các user cùng đang access vào một nội dung cụ thể
