## Overview

Carousel là một dạng layout trong đó chứa nhiều item như card, image,
video, text content cho phép user có thể scroll hoặc click để move out
hoặc in viewport.

## Usage

Trên mobile, carousel chỉ nên có tối đa 2 item nằm hoàn toàn trong
viewport.

Các item nằm trong carousel phải cùng 1 loại.

Khi scroll có hiệu ứng snap item theo page.

Cần có easing để chuyển động khi scroll được mượt mà.

Snap mode is the default mode for the carousel. It snaps to the nearest item when the user stops scrolling.
