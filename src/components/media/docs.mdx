## Overview

Media thể hiện cho image hoặc video được upload và hiển thị trên hệ
thống của Smart Travel, tùy từng môi trường media sẽ có thể có những
action tương ứng khác nhau.

## Usage

Đối với media do user upload lên sẽ cung cấp cho user những quyền như
xóa, drag and drop để thay đổi vị trí.

Đối với media không phải do user upload lên có thể cung cấp những action
như like hoặc share.

Media nên tuân theo những ratio phổ biến như: Square (1:1), Landscape
(4:3), Portrait (3:4), Wide (16:9)

Click vào media sẽ mở full screen, giữ nguyên actions.
