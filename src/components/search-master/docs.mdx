## Overview

Search master chứa những trường input cụ thể liên quan đến mục tiêu
chính của sản phẩm giúp user có thể tìm thấy nội dung cần thiết một cách
nhanh chóng. Xuất hiện ở xuyên suốt các function chính.

## Usage

Search master cần bao gồm tất cả những input cần thiết để filters nhanh
theo nhu cầu của user.

User có thể tự input hoặc sử dụng bộ suggestion dạng auto complete để
trải nghiệm tìm kiếm được dễ dàng nhất.

Sử dụng bộ dropdown list riêng phù hợp với ngữ cảnh sử dụng của user.
