# Range slider

## Overview

Range sliders cho phép user lựa chọn input giá trị theo một khoảng được
quy định sẵn.

<Canvas of={Stories.Default} />

<Controls of={Stories.Default} />

## Usage

- Range sliders cần hiển thị toàn bộ các giá trị mà user được phép nhập.
- Có 2 dạng range sliders là: Continious (user chọn 1 giá trí trong khoảng
  cho phép) và Discrete (User lựa chọn 1 khoảng giá trị trong giới hạn cho
  phép)
- Range slider cần phản hồi giá trị cụ thể ngay lập tức bằng tooltip khi
  user có thao tác kéo handle.
