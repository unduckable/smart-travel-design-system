## Usage

- Sử dụng switch khi muốn bật hoặc tắt ngay lập tức 1 tính năng nào đó.
- Dùng tooltip để chỉ thị cho user chuyện gì sẽ xảy ra khi trigger switch.
- Để giao tiếp tốt hơn với user, có thể sử dụng label kèm theo switch.
