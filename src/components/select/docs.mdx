## Overview

Select là một dạng input mà user có thể lựa chọn một hoặc nhiều option
trong một list item có sẵn. Bao gồm 1 input field và 1 dropdown list.

## Usage

- Select được sử dụng khi có một số lượng lớn option được đưa ra để user
  có thể lựa chọn.
- User có thể chọn 1 hoặc nhiều option.
- Trong dropdown list các option tốt nhất nên chỉ có 1 dòng, nếu content
  quá dài có thể truncate (\...)
- Độ cao của các option trong dropdown list phải bằng nhau
- Khi sắp xếp vị trí của các option trong dropdown list, ưu tiên những
  option hay được lựa chọn lên trên cùng.
- Một số trường hợp user có thể search tại input field để filter options.
