## Overview

Input field bao gồm label và một text field để user có thể nhập thông
tin. Thường xuất hiện trong form và dialogs.

## Usage

- Luôn sử dụng placeholder trong input field, placeholder cần có dạng sát
  với content thật mà user sẽ nhập (vd: Input name thì placeholder nên là
  "Nguyễn Văn A")

- Nếu trường thông tin yêu cầu giới hạn kí tự thì phải có character
  counts.

- Input field sẽ validate ngay khi user blur khỏi input.

- Content bên trong input field không xuống dòng, nếu vượt quá width của
  input sẽ truncate.
