## Overview

Navigation bảo gồm page header và bottom navigation là những components
dẫn dắt user navigate qua lại giữa các page chính trong app Smart
Travel. Bottom navigation bar chỉ nên có từ 3 đến 5 destination page.

## Usage

Các đích đến của bottom navigation sẽ không thay đổi khi user chuyển qua
lại giữa các page.

User có thể access bottom navigation bar từ bất cứ vị trí nào khi sử
dụng app.

Chỉ nên có tối đa 5 destination page.

Không sử dụng đối với các page có độ quan trọng thấp ví dụ như sign in
sign up.

Trên page header có thể bao gồm các action của page, chỉ dùng icon
button và có tối đa 3 actions. Không đưa primary action lên vùng này.
