# Icon

Smart travel sử dụng bộ icon từ iconic.app với style chủ đạo là outline để tránh gây nặng về mặt visual, trong 1 số trường hợp icon từ size 12px trở xuống có thể sử dụng dạng fill.

<img src="/src/assets/media/image15.png" />

<br />
## Icon style

Icon style sử dụng dạng round outline, mềm mại tránh các góc 90 độ, tạo cảm giác thân thiện phù hợp với xu hướng thiết kế hiện đại
Designer có thể bổ sung icon mới vào design system nhưng cần đảm bảo quy chuẩn đã được thể hiện trong STDS UI Kit.
