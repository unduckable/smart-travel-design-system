# Spacing

Việc đặt các đối tượng UI, component trên layout cần phải tuân theo những quy tắc nhất định, việc này nhằm tạo ra một layout có tính quan hệ, cấp bậc rõ ràng và đảm bảo về mặt visual.

## Spacing size

<img src={img14} />

### Spacing size small

Từ 0.5 đến 2 sử dụng cho spacing giữa các element với nhau.

### Spacing size medium

Từ 2 đến 6 sử dụng cho spacing giữa các component.

### Spacing size large

Từ 6 trở lên sử dụng cho spacing giữa các đối tượng container.

## Usage

- Nếu cần spacing lớn hơn để phục vụ về Visual, có thể tăng size của spacing nhưng phải đảm bảo x8 (ví dụ 32-> 40 -> 48 -> 56 -> 64 ...)
- Các value về paddings, margin, spacing và size cần đảm bảo đồng nhất giữa tất cả các giao diện UI của components và toàn bộ sản phẩm.
- Align all the elements to the same spacing within the components or a page.
- Tất cả các components nằm trong 1 trang hoặc 1 container cần được align spacing trên dưới trái phải 1 cách đồng nhất.
